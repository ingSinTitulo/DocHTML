# The `<caption>` element
Defines a table caption

- The defined caption will be visible and by default will appear above the table
- This element must be the first child of the [`<table>`](table.md) element

## Attributes
| Name | Posible values | Notes |
|-|-|-|
| [Global attributes](../first-steps/global-attributes.md) | N/A | `<caption>` supports all global attributes. |

## Example
```html
<!-- Table opening tag -->
    <caption>Registered users data</caption>
<!-- Table closing tag -->
```
