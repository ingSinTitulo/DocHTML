# The `<code>` element
Defines text as source code

- Will be rendered in the browser's default monospace font

## Attributes
| Name | Posible values | Notes |
|-|-|-|
| [Global attributes](../first-steps/global-attributes.md) | N/A | `<code>` supports all global attributes. |

## Example
```html
<code>
function hello () {
    alert('Hello, visitor!');
}
</code>
```
